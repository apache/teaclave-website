TEE client API.

§Feature flags

  • default — enables nothing.
  • macros — re-exports the optee-teec-macros crate as optee_teec::macros, providing the #[plugin_init] and #[plugin_invoke] proc-macro attributes.
  • doc — used for docs.rs to generate docs. It disables native libteec linking.

This type defines a parameter that is not referencing shared memory, but carries instead small raw data passed by value. It is used as a Operation parameter when the corresponding parameter type is one of ValueInput, ValueOutput, or ValueInout.
